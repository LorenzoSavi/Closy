# Closy - Il tuo guardaroba digitale

Closy è un'applicazione web per organizzare il tuo guardaroba in modo semplice ed efficiente. Puoi registrarti, aggiungere capi d'abbigliamento, e creare outfit personalizzati.

## Funzionalità principali
- **Catalogo personale**: Fotografa e organizza i tuoi vestiti.
- **Filtri e categorie**: Filtra i capi per categoria.
- **Interfaccia moderna**: Design responsive e intuitivo.

## Requisiti
- .NET 8
- .0
- SQLite

## Come eseguire il progetto
1. Clona il repository:
   ```bash
   git clone https://github.com/LorenzoSavi/Closy.git
   cd Closy
   ```
2. Ripristina i pacchetti NuGet:
   ```bash
   dotnet restore
   ```
3. Avvia l'applicazione:
   ```bash
   dotnet run
   ```
